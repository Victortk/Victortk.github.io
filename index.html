In computing, memory refers to a device that is used to store information for immediate use in a computer or related computer hardware device. There are two main kinds of semiconductor memory, volatile and non-volatile.

Non-volatile memory (NVM) or non-volatile storage is a type of computer memory that can retrieve stored information even after having been power cycled. In contrast, volatile memory needs constant power in order to retain data.
